Project info:

The Fallen One is a boss rush game where you fight the Four Horsemen of the Apocalypse

This game was made in 2024 and was my final project of my 2nd year

What I've done:

In this project I've imported my previously made audio system and animation system that I've made in TerraTrash

I've created a particle system and implemented particle in the game to make it feel more alive

I've improved on my camera system to allow more control for the dev to make cutscene and be able to have camera shake

I've made the main menu and all it's interaction

I've made the ingame health bar HUD

I've worked almost on the entierty on the last boss

I've made custom shaders in GLSL to animate the black hole and water in the last stage and to give a glowing effect on selected button in the main menu.
