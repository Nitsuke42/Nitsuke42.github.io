Project info:

TerraTrash is a tabletop game where you two player localy play against each other in a bloody american football match.

This game was made in 2023 and was my first trimester project of my 2nd year.

What I've done:

I've done the isometric grid system and implemented a matrix conversion for translating the mouse position on screen to the grid

I've done a pathfinding system to highlight the path taken by the units

I've made a animation system which handled the spritesheet of animation

I've made a sound system that used a audio pool to find a avalaible

I've made a camera system that you can use to move the camera and zoom in/out
