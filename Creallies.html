Project info:

Creallies is a collection game where you scan QR code that were spreed out in the school, to enconter Allies and chose to capture them or fight them.

This game was made in 2024 and was my first trimester project of my 3rd year.

What I've done:

In this project I've done the scanning of QR code to spawn a random Allies in function of the zone in which the QR code was scanned.

I've made the capture section of the game where you play a mini game that allow you to catch the meeted Allies.

I've also made a shader for one of the legendary Allies we created.
