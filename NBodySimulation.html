Project info:

N-Body Simulation DLL is a small personal project I've made, which is a C++ library of a N-Body simulation ready to be implemented in any C, C++, or C# project

This project was made in 2025 on my free time, after I've discovered what a n-body simulation was and wanted to do my own library for it

What I've done:

I've created a library in C++ that I imported in Unity. The library take care of managing the simulation and updating all celestial body position, while Unity is used for the render part and to call the library to advance to the next step of the simulation

I've also added a way of adding planet to the simulation with Unity by giving a velocity and mass to the celestial body so the library can add it to the simulation
